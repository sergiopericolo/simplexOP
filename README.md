# SimplexOP

SimplexOP is a web-based utility designed to perform **Elementary Row Operations** on matrices, specifically optimized for the **Simplex Method** in linear programming.

## Purpose
The tool serves as a computational aid to execute manual iterations of the Simplex algorithm.It eliminates mental calculation errors during row pivoting, ensuring precision in optimization tasks.

## Core Functionality
* **Vector Operations**: Performs the operation $R_1 \pm (m \cdot R_2)$ across a row of up to 10 elements.
* **Dual Representation**: Supports both floating-point decimals and **exact fraction** arithmetic to prevent rounding errors.
* **Dynamic Scaling**: Adjusts the input grid size based on the number of decision variables and slack variables in the tableau.
* **Input Flexibility**: Handles integers, decimals, and fractional inputs directly.
